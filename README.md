## Getting Started

1. Install node_modules dependencies`npm install or yarn install`

2. Run the development server:

```bash
npm run dev
```

3.Run test

```bash
npm run test
or 
npm run test:watch  
# depending on whether you want to update anytime
```

Go to [Local Dev Link](http://localhost:3000)
